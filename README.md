## 스프링을 사용한 게시판 만들기

  회사 신입으로 들어와서 두 번째로 받은 과제 입니다.

  스프링을 사용하여 게시판 만들기.

## 구현 목표 

  1. 게시글 목록
  2. 등록(CREATE)
  3. 읽기(READ)
  4. 수정(UPDATE)
  5. 삭제(DELETE)
  6. 페이징
  7. 검색
  8. 첨부파일 2개 이상

## 개발 환경

  - STS 3.9
  - DBeaver
  - 이클립스
  - MySQL, MyBatis
  - open JDK 1.8
